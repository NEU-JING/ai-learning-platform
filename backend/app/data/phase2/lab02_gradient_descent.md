# 实验2 - 手写梯度下降与正则化对比

## 实验目标

1. 从零实现梯度下降（含偏置项）
2. 实现L2正则化（岭回归）
3. 实现L1正则化（Lasso近似）
4. 对比三种方法的参数分布和泛化能力

## 背景

你有一个含50个特征的数据集，但只有20个特征与目标相关。你需要验证：
- 无正则化：过拟合，50个特征权重都不为零
- L2正则化：权重被压缩但不为零
- L1正则化：自动将无关特征权重推为零

## 要求

### Part 1: 生成含冗余特征的数据

```python
# 20个有用特征 + 30个噪声特征
# y = X[:, :20] @ true_w + noise
```

### Part 2: 实现梯度下降

- 均方误差损失
- 偏置项单独处理（不正则化）

### Part 3: 实现L2正则化

- 损失 = MSE + alpha * ||w||²
- 偏置项不参与正则化

### Part 4: 实现L1正则化

- 损失 = MSE + alpha * ||w||₁
- 使用软阈值(soft thresholding)处理L1的不可微点

### Part 5: 对比分析

- 三种方法的非零权重数
- 训练集/测试集MSE
- 权重分布对比

## 提示

- L1的梯度用subgradient：sign(w)
- 偏置项加在w[0]，正则化时跳过w[0]
- 学习率建议0.001，迭代5000次
